# clinic — модульное CLI-приложение на Dart

Консольное приложение для автоматизации работы клиники с хранением данных в SQLite.

## Предметная область

Система автоматизации клиники:

- пациенты (`Patient`);
- врачи (`Doctor`);
- должности (`Position`);
- записи на приём (`Appointment`);
- медицинские карты (`MedicalCard`);
- услуги (`Service`);
- оплаты (`Payment`);
- расписание врачей (`DoctorSchedule`).

## Архитектура и структура папок

Проект разделен на модули по слоям:

```text

        appointment.dart
        medical_card.dart

Связи между сущностями

Реализованы связи между таблицами:

врач связан с должностью через position_id;
запись на приём связана с пациентом через patient_id;
запись на приём связана с врачом через doctor_id;
медицинская карта связана с пациентом и врачом;
оплата связана с записью и услугой;
расписание связано с врачом.

Все связи реализованы через FOREIGN KEY.

Валидация

Реализованы проверки:

Обязательный текстовый ввод

Проверяется:

строка не пустая;
строка не пустая после trim().
Числовые поля

Проверяется:

число больше 0.

Используется для:

зарплаты;
стоимости услуг;
суммы оплаты.
 Проверка даты

Проверяется корректность через:

DateTime.tryParse()

Поддерживается формат:

YYYY-MM-DD
